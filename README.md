<h1><p>SHOOTING GAME IS DEVELOPED USING HTML, CSS, AND JAVASCRIPT. TALKING ABOUT THE GAMEPLAY, IT IS A SIMPLE GAME IN WHICH THE PLAYER HAS TO SHOOT ALL THE TOWARDS MOVING RED BALLS.
